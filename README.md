### Stages de Jenkinsfile

1. **Build**
   Construye la imagen Docker para la aplicación.

2. **Test**
   Ejecuta pruebas automatizadas dentro de un contenedor Docker.


3. **Deploy**
   Despliega la imagen Docker en Google Cloud.
